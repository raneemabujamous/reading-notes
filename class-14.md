 ### CSS 
* `transform` : This property allows you to rotate, scale, move, skew, etc., elements
* `The transform-origin` property allows you to change the position of transformed elements.
This property must be used together with the transform property.<br>
** initial	Sets this property to its default value **<br>
** inherit	Inherits this property from its parent element **
*  `The transform-style` property specifies how nested elements are rendered in 3D space.`Note`: This property must be used together with the transform property.
* `transitions` allows you to change property values smoothly, over a given duration.Mouse over the element below to see a CSS transition effect
1-transition<br>
2-transition-delay<br>
3-transition-duration<br>
4-transition-property<br>
5-transition-timing-functio<br>
* `transition-timing-function` property specifies the speed curve of the transition effect.<br>
1-`ease -` specifies a transition effect with a slow start, then fast, then end slowly (this is default)<br>
2-`linear` - specifies a transition effect with the same speed from start to end<br>
3-`ease-in `- specifies a transition effect with a slow start<br>
4-`ease-out` - specifies a transition effect with a slow end<br>
5-`ease-in-out `- specifies a transition effect with a slow start and end<br>
6-`cubic-bezier`(n,n,n,n) - lets you define your own values in a cubic-bezier functionv
* `The transition-delay` property specifies a delay (in seconds) for the transition effect.
